# Release Notes 0.0.0.27

## Main changes
- Introduced the `V2` UI structure for the main window and admin area.
- Reworked the main toolbar into clearer user-facing sections: search, active teacher, settings, and administration.
- Grouped admin workflows into `Main`, `Import & Sync`, and `System`.
- Added helper text across the main workflows so users can understand the purpose of each screen faster.

## UX improvements
- Simplified labels in report, import, synchronization, system, and file-related actions.
- Improved report readability with guidance text, color legend, and better selection feedback.
- Improved teacher-selection, password, import-preview, batch-import, and synchronization-conflict dialogs.
- Unified button density and helper-text styling across the main admin screens.

## Admin and system screens
- Reorganized action bars so primary actions appear in more consistent places.
- Clarified local sync and Internet sync screens with step-by-step guidance.
- Improved settings and log screens with clearer grouping and more explicit action names.
- Added more consistent visual spacing for group boxes, hints, and action rows.

## Versioning
- Version sync is aligned with the latest commit marker.
- Application version is now synchronized to `0.0.0.27`.
